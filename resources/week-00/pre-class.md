# Gate 0：Week 0 准备检查

Week 0 没有机制前置周，但需要确认你能运行本仓库并阅读二维数组。

## 进入前的最小检查

1. 运行 `uv sync --group dev`。
2. 用一句话说明二维数组 `(3, 2)` 有三行、两列；暂时不要求解释 attention。
3. 打开 [Week 0 导航页](../../weeks/week-00/README.md)，说出本周结束时要为
   Week 1 提供的能力。

## 进入条件

- 环境命令可运行；
- 能读出一个二维 shape；
- 知道本周目标是追踪信息如何变化，而不是提前实现 attention。
