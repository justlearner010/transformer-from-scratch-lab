# 课前引导

## 进入本周前应能回答

1. 矩阵乘法的内维度为什么必须相等？
2. 一个长度为 `n`、特征维度为 `d` 的 token 序列 shape 是什么？
3. softmax 输出为何能被看作权重？

## 常见误解

- Q、K、V 不是三份固定的词向量，而是同一输入经不同可学习投影得到的表示。
- attention 权重不是“词的重要性”的绝对分数，而是当前 query 对所有 key 的相对分配。
- softmax 不是简单地把数值压到 0 和 1 之间；它必须沿正确的轴归一化。

## 学习目标

完成本周后，应能不看代码写出 `QK^T / sqrt(d_k)` 的 shape，并解释为何对每个 query 的权重行求和为 1。
